## 应用功能介绍｜人生记录（Life Tracker）

### 1. 人生格子日历

* 以**生日为起点**，按天生成完整人生时间轴
* 支持 **不分组 / 按年份 / 按月份** 三种视图
* **打开应用自动跳转到今天的格子**
* 已过去、今天、未来日期自动区分显示
* 点击任意格子即可查看或编辑当天内容

---

### 2. 人生记录统计

* 自动统计并实时展示：
  * 已度过天数
  * 已记录天数
  * 精彩日数量
* 支持设置**人生终点年龄**（默认 80 岁，可修改）
* 提供**人生进度条**，直观显示当前人生进度百分比
* 所有统计数据随记录自动更新，无需手动维护

---

### 3. 每日多条记录

* 每一天支持**添加多条记录**
* 每条记录支持：
  * 文本内容
  * 图片或其他文件附件
* 记录支持**编辑 / 删除**
* 记录以列表形式展示，结构清晰

---

### 4. 精彩日标记

* 可将某一天标记为“精彩日”
* 提供多种颜色用于区分不同类型的重要日子
* 精彩日会在日历中高亮显示，便于快速定位与回顾

---

### 5. 备忘录功能（跨日期）

* 支持创建**跨日期的备忘事项**
* 备忘录支持设置：

  * 状态（待跟进 / 处理中 / 已完成）
  * 开始日期、结束日期
  * 优先级、备注
* 备忘事项会自动映射到对应日期的日历格子中显示

---

### 6. 本地存储

* 所有数据仅保存在本地
* 无需账号登录
* 无云端同步
* 不依赖网络，离线可用

---

### 7. 桌面应用

* 基于 **Electron** 构建
* 支持 Windows 桌面运行
* 打开即用，数据长期保存在本机

---

## 下载与安装指南

### 方式一：直接运行（已打包版本）

1. 下载应用压缩包
2. 解压后，进入应用目录
3. 双击运行可执行文件（`.exe`）
4. 首次打开需设置生日，设置完成后自动进入日历界面

> 说明：
>
> * 数据将保存在本地
> * 卸载应用前如需保留数据，请备份用户数据目录

---

### 方式二：源码运行（开发者）

1. 安装 Node.js（建议 LTS 版本）
2. 解压或克隆项目代码
3. 在项目根目录执行：

   ```bash
   npm install
   ```
4. 启动应用：

   ```bash
   npm start
   ```
5. 如需打包 Windows 应用：

   ```bash
   npm run pack
   ```

---

### 使用提示

* 首次使用必须设置生日，
* 人生终点默认设置为80岁
* 建议定期自行备份本地数据目录
